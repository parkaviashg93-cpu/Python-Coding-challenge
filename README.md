Feedback Analyzer (Python)
A simple Python-based text analysis tool that processes user feedback and extracts useful insights using basic string manipulation techniques.
This project demonstrates fundamental Python programming concepts such as string operations, text processing, user input handling, and simple data analysis.
________________________________________
 Project Overview
The Feedback Analyzer takes user feedback as input and performs several analyses on the text. It helps demonstrate how raw text data can be processed and analyzed using Python.
This project is ideal for beginners learning:
•	Python fundamentals
•	String manipulation
•	Basic text analytics
•	Writing structured Python programs
________________________________________
 Features
•	Accepts user feedback as text input
•	Counts the total number of words
•	Detects keyword occurrences (e.g., "data")
•	Extracts the first 5 characters and last 5 characters
•	Reverses the feedback string
•	Demonstrates basic text processing techniques
________________________________________
 Technologies Used
•	Python 3
•	Jupyter Notebook
•	Basic Python string functions

 How It Works
The program performs the following steps:
1.	Takes feedback input from the user.
2.	Splits the text into words to count the total number.
3.	Searches for a specific keyword (data) in a case-insensitive way.
4.	Extracts portions of the text using string slicing.
5.	Reverses the full feedback string.
